# Página Motivadora para Mi Amor 💜

Esta es una página web sencilla y emotiva creada para motivar y recordar a mi novia lo mucho que la amo y creo en ella.

## Cómo Ver la Página

### Opción 1: Abrir Localmente

1. Descarga o clona este repositorio en tu computadora.
2. Abre el archivo `index.html` en tu navegador web (Chrome, Firefox, etc.).
3. ¡Disfruta la página!

### Opción 2: Subir a GitHub Pages

1. Crea un nuevo repositorio en GitHub (público o privado).
2. Sube todos los archivos (`index.html`, `styles.css`, `script.js`, `README.md`) al repositorio.
3. Ve a la pestaña "Settings" del repositorio.
4. Desplázate hacia abajo hasta "Pages" en el menú lateral.
5. En "Source", selecciona "Deploy from a branch".
6. En "Branch", selecciona "main" (o la rama principal) y "/ (root)".
7. Haz clic en "Save".
8. Espera unos minutos y GitHub te dará una URL como `https://tuusuario.github.io/nombre-del-repo/`.
9. Comparte esa URL con tu amor para que pueda verla desde cualquier dispositivo.

## Personalización

### Cambiar Textos

- Abre `index.html` en un editor de texto.
- Busca las secciones entre `<h1>`, `<p>`, etc.
- Edita el texto dentro de las etiquetas para personalizar mensajes, frases o títulos.
- Guarda el archivo y recarga la página.

### Cambiar Colores

- Abre `styles.css`.
- Busca propiedades como `background`, `color`, etc.
- Por ejemplo, para cambiar el gradiente de fondo, edita la línea:
  ```css
  background: linear-gradient(135deg, #2b0a3d, #4b1d6d, #7b2cbf);
  ```
- Reemplaza los códigos de color (#hex) con otros tonos morados o los que prefieras.
- Para colores suaves, usa tonos como #e0b3ff para acentos.

### Cambiar Imágenes

- En `index.html`, busca las etiquetas `<img src="...">`.
- Reemplaza las URLs con tus propias imágenes (sube a un servicio como PostImage o Imgur para obtener URLs).
- Asegúrate de que las imágenes sean apropiadas y de buena calidad.

### Ajustar Animaciones

- En `styles.css`, busca `@keyframes` para editar animaciones.
- En `script.js`, ajusta tiempos o frecuencias (ej. `setInterval(createHeart, 3000);` para corazones flotantes).

## Tecnologías Usadas

- HTML5
- CSS3 (con animaciones y gradientes)
- JavaScript Vanilla (sin frameworks)

## Notas

- La página es responsive, así que se ve bien en celular y computadora.
- Las animaciones son sutiles para mantener un tono cálido y emotivo.
- Si tienes problemas, asegúrate de que todos los archivos estén en la misma carpeta.

¡Espero que esta página haga sonreír a tu amor! 💜
